# 🎲 Truth or Dare Game

A fun and interactive **Truth or Dare** web game built using **HTML, CSS, TypeScript, and JavaScript**. Players can choose **Truth**, **Dare**, or let the game randomly decide. The game displays random questions from a JSON dataset and includes a countdown timer to make gameplay more engaging.

## 🌐 Live Demo

👉 **Live Website:** https://kriti-kumari221.github.io/Truth_Dare_Game/

---

## ✨ Features

- 🎯 Random Truth Questions
- 🎯 Random Dare Challenges
- 🎲 Random Mode (Truth/Dare)
- ⏳ Countdown Timer
- 🔒 Buttons are disabled while the timer is running
- 📂 Questions loaded dynamically from a JSON file
- 📱 Responsive and clean UI

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Kriti-kumari221/Truth_Dare_Game.git
```

### Navigate to the Project

```bash
cd Truth_Dare_Game
```

### Install Dependencies

```bash
npm install
```

### Compile TypeScript

```bash
tsc
```

### Run the Project

Open `index.html` using **Live Server** in VS Code.
